磁盘管理常用命令

df 命令

df命令用于显示有关每个文件所在的文件系统的信息，或者默认为所有文件系统。
命令格式：df 选项 参数

选项 解释
- a 显示全部文件系统
- k 指定区块大小为1024字节；
- i 显示节点信息
-t + 类型 显示指定文件系统类型的磁盘空间使用情况
- T 显示文件系统类型
-x + 类型 显示除指定文件系统类型的磁盘空间使用情况
- h 以可读性较高的方式来输出信息

du 命令

以递归方式汇总每个文件的磁盘使用情况。

命令格式：du 选项 参数

选项 解释
- a 显示每个文件的大小
- b 以byte为单位显示文件大小
- c 同时显示文件大小和
- k 以KB为单位输出
- m 以MB为单位输出
- s 只显示文件大小和

dd命令

dd命令用于复制文件并对原文件的内容进行转换和格式化处理。

命令格式：dd 选项

选项 解释
bs= 每次读写块的大小（字节）
if= 输入文件名
of= 输入文件名
skip= 一开始读取时，跳过指定的区块数
seek= 一开始输出时，跳过指定的区块数
conv= 指定文件转换的格式

fdisk命令

用于观察硬盘实体使用情况，也可对硬盘分区。分区使用方式在上面已经使用。

mkfs命令

mkfs命令（make file system）用于在设备上（通常为硬盘）创建Linux文件系统。只是将分区格式化。

命令格式：mkfs 选项 参数

- t 指定建立何种文件系统
- v 显示版本信息与详细的使用方法
- V 显示简要的使用方法

在这里我们之前建立分区的最后已经格式化为xfs系统了，所以格式化失败。
