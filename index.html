数据库系统概论

发表于 2019-11-03 分类于 大学课程 阅读次数：

[TOC]

一 绪论

1.1 数据库系统概述

1.1.1 数据库的基本概念

  1. 数据：描述事物的符号记录称为数据，数据的含义称为数据的语义，数据与其语义是不可分的。
  2. 数据库：数据库是长期存储在计算机内、有组织的、可共享的大量数据的集合。数据库中的数据按照一定的数据模型**组织、描述和储存，具有较小的冗余度、较高的数据独立性和易扩展性，并为各种用户共享**。
  3. 数据库管理系统：是位于用户与操作系统之间的一层数据库管理软件。
  4. 数据库系统：数据库系统是由数据库、数据库管理系统、应用程序和数据库管理员组成的存储、管理、处理和维护数据的系统。

发表于 2019-11-03 分类于 大学课程 阅读次数：

目标

  1. 完成网络文档。
  2. 在路由器和交换机上执行基本的设备配置。
  3. 检验连接并排除故障。

要求

  1. 提供地址分配表中缺少的信息
  2. 命名路由器 College 和第二个交换机 Class-B。您将无法访问 Class-A。使用 cisco 作为所有线路的用户 EXEC 模式密码。
  3. 使用 class 作为特权 EXEC 模式密码。
  4. 加密所有明文密码。
  5. 配置相应的标语。
  6. 根据地址分配表为所有设备配置编址。
  7. 记录接口及其描述，包括 Class-B VLAN 1 接口。
  8. 保存配置。
  9. 检验所有设备之间的连接。所有设备应能 ping 通任何其他设备。
  10. 排除故障并记录问题。
  11. 实施启用和检验完整端到端连接所需的解决方案。
